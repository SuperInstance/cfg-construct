# cfg-construct

Control flow graph construction from basic blocks with dominance analysis for compiler infrastructure.

## Features

- **Basic blocks** — Instructions, predecessors, successors
- **CFG construction** — Block creation, edge insertion, reverse postorder
- **Dominance analysis** — Iterative dominance computation with immediate dominators
- **Dominance frontier** — Frontier computation for SSA construction
- **Natural loop detection** — Back edge identification and loop body computation
- **Zero dependencies** — Pure `std` implementation
